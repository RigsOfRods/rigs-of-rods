  Rigs of Rods project (www.rigsofrods.org)
  Code template for MyGUI's Layout Editor
  
  Installation:
  
  To install this template, copy the .xml + .cpp + .h files into:
  [MyGUI]/Media/Tools/LayoutEditor/CodeTemplates/ directory.
  In MyGUI 3.2.0, the layout XML file name is hardcoded to "BaseLayoutCPP.xml", 
  so you need to replace the default file with this one.
  
  Formatting guidelines:
  
  1. Widget naming
  All widgets should have lowercase_names_with_underscores. Scheme example:
  * rig_editor_menubar : Menubar
  * item_file : Menubar item File
  * file_popup : Menubar popup File
  * file_popup_item_open
  
  2. Code generation
  Panel name: CamelCaseLayout, example: RigEditorMenubarLayout
  Namespace: any value, not used in template
            
  Thank you for your interest in Rigs of Rods!

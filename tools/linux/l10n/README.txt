This README contains the usual workflow to extract, translate and set up translations for delivery.

Important: Read the indiviual scripts for further information and setup.
You have to edit to_transifex.sh in particular in order to push to Transifex.

extract_pot.sh
setup_transifex.sh
from_transifex.sh
merge_old_translations.sh
to_transifex.sh
<wait for community to translate>
from transifex.sh
compile_mo.sh

Done!
The languages folder can now be put in the bin folder of RoR (same level as the RoR executable).
